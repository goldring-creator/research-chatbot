4

앱 실행

Mac

압축 푼 ResearchChatbot.app 을
우클릭 → 열기 → 열기

⚠️ "확인되지 않은 개발자" 경고가 떠도 정상입니다. (처음 한 번만 우클릭 열기)

⊞ Windows

폴더 안 ResearchChatbot.exe 더블클릭
경고 시 추가 정보 → 실행

⚠️ "Windows의 PC 보호" 경고가 떠도 정상입니다. (처음 한 번만)

5

무료 NVIDIA API 키 발급 후 입력

처음 실행하면 키 입력 화면이 나옵니다. 아래 순서로 무료 키를 받아 붙여넣으세요.

  1. 앱의 [NVIDIA 사이트 열기] 클릭, 또는 여기를 눌러 NVIDIA 사이트 열기 → build.nvidia.com
  2. 우측 상단 Login → 구글·이메일로 로그인
  3. 빨간 줄이 보이면 Verify(계정 인증) 클릭
  4. 아무 모델 열기 (예: deepseek 검색)
  5. Build 탭 → Generate API Key 클릭
  6. 만들어진 nvapi-... 키 Copy
  7. 앱 입력칸에 붙여넣고 [저장하고 시작]
📷 참고 화면 (순서대로)

※ 발급된 키가 보이는 화면은 보안을 위해 첨부하지 않았습니다. ‘Copy’로 복사한 nvapi- 키를 앱에 붙여넣으세요.

🔒 API 키는 비밀번호처럼 다루세요. 키는 본인 컴퓨터에만 저장되며 외부로 전송되지 않습니다.
💡 한 번 입력하면 다음부터는 바로 채팅창으로 시작합니다. 대화·답변은 문서 / Documents / ResearchChatbot 폴더에 저장됩니다.
